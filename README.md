## Project Title: Extrovert vs Introvert Behavioural Pattern

A data analysis project exploring behavioural differences between extroverts and introverts using a Kaggle dataset. The goal was to understand how personality types influence social habits such as posting frequency, time spent alone, social event attendance, and friend circle size.

## 📌 Project Overview

- **Motivation:**  
  I chose this dataset to explore how personality traits (extroversion vs introversion) influence social and digital behaviour. It’s a relatable topic with real-world relevance in psychology, social media, and human interaction.  

- **Objective:**  
  To analyze and compare behavioural patterns between extroverts and introverts, focusing on:  
  - Social activity levels  
  - Online engagement (posting frequency)  
  - Time spent alone  
  - Social event participation  
  - Size of friend circles  

- **Learning Outcomes:**  
  - Improved data cleaning skills using Excel  
  - Gained hands-on experience building dashboards in Tableau  
  - Learned how to translate raw data into meaningful insights and storytelling  
  - Strengthened ability to identify behavioural patterns through visualization

## 📁 Dataset

- **Source:**  
  https://www.kaggle.com/datasets/rakeshkapilavai/extrovert-vs-introvert-behavior-data  

- **Dataset File:**  
  `personality_datasert.csv`

- **Key Features/Columns Used:**  
  - Personality Type (Extrovert / Introvert)  
  - Post Frequency  
  - Time Spent Alone  
  - Going Outside  
  - Social Event Attendance  
  - Friends Circle Size  

- **🧼 Data Cleaning Steps:**  
  - Rounded all decimal values to 0 decimal places  
  - Removed leading and trailing whitespaces  
  - Checked and handled duplicates  
  - Ensured correct data types for all columns

💡 Analysis & Visualizations

Going Outside
	•	Extroverts go outside more frequently
	•	Introverts show a rapid drop after a few outings

Post Frequency
	•	Introverts are less active in posting (0–2 range), suggesting passive engagement
	•	Extroverts post more consistently across higher ranges

Time Spent Alone
	•	Introverts dominate higher time-alone categories
	•	Extroverts are more present in lower ranges

Friends Circle Size
	•	Extroverts generally maintain larger social circles
	•	Introverts tend to have smaller, closer groups

Social Event Attendance
	•	Extroverts attend more events across all ranges
	•	Introverts cluster heavily in low attendance

Personality Distribution
	•	51.4% Extroverts
	•	48.6% Introverts

⸻

📌 Conclusion
	•	Extroverts are more socially and visibly active across most metrics
	•	Introverts engage differently, often more passively rather than visibly
	•	Posting frequency alone does not fully represent online engagement
	•	Personality traits strongly influence both digital and real-world behaviour
